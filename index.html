HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The title element defines the title of the webpage
  • The base element defines the base URL for a webpage
  • The meta element defines metadata such as the character set, description, keywords, author, and veiwport

CSS

  • The style element defines the CSS styles associated with a website
  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • The HTML class attribute specifies one or more class names for an element

GIT

  • git status: checks what branch we are currently on
  • git checkout -b (branch-name): creates a new branch and switches to it
  • ls: lists files in current folder1
  • mkdir: make new directory
  • cd:changes into different directory
  • touch: create a new file
  • pwd: print working directory
  • mv: rename file
  • rm: remove file
  • history: show all commands you have run

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Array is single variable that is used to hold a group of data that is typically related in some way.
  • Array can store different types of data, such as strings, numbers, and so on.
